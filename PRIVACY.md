# Privacy Policy for ChatGPT Companion

ChatGPT Companion helps users discuss the webpage they are viewing with ChatGPT.

## Data the extension handles

The extension may handle the following data:

- Page titles and page URLs
- Selected text from webpages
- Clicked link text and URLs
- Generated ChatGPT prompts
- Saved ChatGPT conversation URLs
- Extension settings and prompt templates
- Chrome profile sign-in status and email, used only to show whether Chrome sync may be available

## How the data is used

This data is used only to provide the extension's single purpose: creating, continuing, saving, and restoring ChatGPT discussions about webpages selected by the user.

The extension does not use this data for advertising, analytics, profiling, sale, or unrelated purposes.

## Data storage

Discussion sessions, tab mappings, generated prompts, saved ChatGPT conversation URLs, settings, and prompt templates are stored in Chrome extension storage on the user's device.

If the user enables Cloud Sync, settings and prompt templates may be stored using Chrome Storage Sync. Discussion sessions and page discussion history are not synced through Cloud Sync.

## Data sharing and transfer

ChatGPT Companion does not collect or transmit user data to the developer's servers.

When the user starts or continues a discussion, the generated prompt may include page information, clicked link information, or selected text. That prompt is inserted into ChatGPT. ChatGPT is provided by OpenAI and is subject to OpenAI's own terms and privacy policy.

The extension includes a voluntary support link to Buy Me a Coffee. Opening that link leaves the extension and loads the Buy Me a Coffee website, which is subject to its own privacy practices.

## Limited Use

The use of information received from Chrome extension permissions complies with the Chrome Web Store User Data Policy, including the Limited Use requirements.

The extension uses personal or sensitive user data only to provide or improve its single purpose and user-facing features. The extension does not use or transfer user data for personalized advertising, retargeting, or unrelated purposes.

## Human access

The developer does not read, review, or access user discussion data.

## User control

Users can clear saved discussion sessions from the extension settings page.

Users can disable Cloud Sync from the extension settings page. Users can also remove all extension data by uninstalling the extension or clearing extension storage in Chrome.

## Contact

For privacy questions, contact the developer through the support information listed on the Chrome Web Store page.
